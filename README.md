# Jaculus-libraries
Jaculus repository for all libraries

## How to use

### Install

```bash
make install
```

### Generate web

```bash
make generate
```

### Serve web

```bash
make serve
```

without livereload
```bash
make serve-no-livereload
```

more info
```bash
python3 jaculusLibraries.py --help
```
